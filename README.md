# SaveAllFiles

A BetterDiscord plugin that Allows you to download all files from a message to a folder at once without prompts

<img height="400" alt="Capture" src="https://github.com/user-attachments/assets/adda2d75-98a8-4b75-b046-80338b6b9dec" />


## What it does

Right-click any message with files and you'll see a "Save All Files" option. Click it and all the attachments and embedded images/videos get saved to a folder of your choice. Works with regular file attachments, embedded images, thumbnails, and videos.

## Installation

1. Download `SaveAllFiles.plugin.js` and drop it into your BetterDiscord plugins folder
2. Enable the new plugin in discord
3. If it asks, let it download the BDFDB Library (it needs this to work)

## Settings

You can customize a few things in the plugin settings:

**Save Folder Path** - Where you want files saved. If you don't set one, it'll use your Downloads folder.

**Context Menu Position** - Where the "Save All Files" button shows up in the right-click menu. You can put it at the top, bottom, or near other options like Copy Text or Edit.

**Button Label** - Change what the button says. Default is "Save All Files" but you can make it whatever you want.

**Show Folder Link in Notification** - Turn this on if you want a clickable link in the success message that opens your save folder.
